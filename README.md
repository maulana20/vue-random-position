# vue-random-position

## Getting Started

Sebagai pembelajaran penggunaan vue

sumber https://github.com/hadihammurabi/vue-acak-kelompok

Langkah sebelum memulai
```bash
npm install
```

Jalankan vue melalui npm
```bash
npm run serve
```

buka pada halaman http://localhost:8081
